════════════════-[ About ]═══════════════════

I've stated this char back in 2005, but I lost it on a HD crash and I think that it were lost forever.

But this year I saw a screenshot at Alex's website, Mugenarok.com.br, with it and I remembered that him
were the only person who had it besides me. So, I have a BIG thank you to Alex.

Eder, a friend of On.Off, asked me if I would like to code a creation of him and once I put my eyes
on this char...I realized that it could turn into something really funny.

I`ve tried to make it as original and fun as I can do, with cartoon sounds, original moves

	This char is a 3 button char.

	X= weak attack
	Y= medium attack
	Z= hard attack

	Down+Z = Launcher

	* Move can be done on air
	# Move MUST be done on air
	• Move is a follow-up starter
	└ Move must be done chaining from the follow-up starter above it
